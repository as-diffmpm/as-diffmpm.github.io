TL;DR: CollidingGS enables collisions with arbitrarily shaped objects.


Abstract

System identification involving the geometry, appearance, and physical properties from video observations is a challenging task with applications in robotics and graphics. Recent approaches have relied on fully differentiable Material-Point-Method (MPM) and rendering for simultaneous optimizations of these properties. However, they are limited to simplified object-environment interactions with simple planar colliders and fail in more realistic scenarios where objects collide with non-planar surfaces. Available MPM-based simulators that support such complex interactions lack differentiability for gradient-based optimizations. We propose AS-DiffMPM, a differentiable MPM framework that enables physical property estimation with arbitrarily shaped colliders. Our approach extends existing methods by incorporating a differentiable collision handling mechanism, allowing the target object to interact with complex objects while maintaining end-to-end optimization. We show AS-DiffMPM can be easily interfaced with various novel view synthesis methods as a framework for differentiable system identifications.


Colliding GS = Collisions-enabled MPM + 2DGS-based collider



Given multi-view input images, we segment out the continuum object and rigid body collider for reconstruction. Then, we extract particles and advect them using our AnyShape Differentiable Material Point Method (AS-DiffMPM). The particle trajectories are mapped back to rendering primitives for photorealistic and physically plausible animation.



System Identification Results


System identification involves estimating the physical parameters of objects from visual observations. Our AS-DiffMPM can be seamlessly integrated with various rendering methods (e.g., 2DGS, DVGO, MDyn-3DGS.) for the task at hand.
